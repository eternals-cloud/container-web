# Container

### Version

> 1.0

## Description

"Container" is a web app for managing Docker containers. It features user authentication and real-time monitoring, simplifying container management for developers and admins.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
